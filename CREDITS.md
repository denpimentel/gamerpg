# Créditos dos Assets

Este repositório usa apenas assets gratuitos de terceiros. Os arquivos originais estão em
`assets/` (não versionados na web) e as versões processadas (chroma key, spritesheets
montados, recortes) em `public/assets/`.

## Demo 32×32 — Vila de Camwood

- **Liberated Pixel Cup (LPC) Base Assets** — tiles e sprites (soldier, slime, terreno,
  árvores, lagoa).
  - Fonte: https://opengameart.org/content/liberated-pixel-cup-lpc-base-assets-sprites-map-tiles
  - Licença: **CC-BY-SA 3.0 / GPL 3.0** (dual). Autores listados em
    `assets/32/LPC Base Assets/CREDITS.TXT` (Lanea Zimmerman "Sharm", Stephen Challener
    "Redshrike", entre outros). Uso comercial permitido com atribuição + share-alike.

## Demo 64×64 — Ilha de Elmsong

- **Tiny Swords** — Pixel Frog. Terreno, água/espuma, árvores, guerreiro, goblin, ovelha, deco.
  - Fonte: https://pixelfrog-assets.itch.io/tiny-swords (obtido via mirror público no GitHub:
    `6-team/tiny-swords`)
  - Licença: **CC0** (domínio público; uso comercial livre, sem atribuição obrigatória).

- **Reiner's Tilesets** — Freya (machado), esqueleto espadachim, árvores, arbustos,
